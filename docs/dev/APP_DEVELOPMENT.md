### 1.2 与网站的同步机制

| 模块 | 网站状态 | APP适配策略 |
|------|---------|------------|
| 首页布局 | 手机版适配中 | 复用手机版设计 |
| 帖子展示 | 田字格布局 | 保持一致的卡片样式 |
| 搜索功能 | 移动端隐藏 | APP需保留搜索入口 |
| 用户系统 | 基础版 | APP需完整的登录态管理 |
| 圈子系统 | 5个圈子 | 与网站完全同步 |
| 热度排序 | 实时计算 | 复用网站API |
| 积分系统 | 基础版 | 需实时同步积分变动 |

## 三、API接口清单（与网站共用）

### 3.1 用户相关

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 登录 | POST | /api/login | 邮箱+密码 |
| 注册 | POST | /api/register | 需选择圈子 |
| 用户信息 | GET | /api/users/:id/profile | 含积分等级 |
| 更新信息 | PUT | /api/users/:id | 修改资料 |
| 排行榜 | GET | /api/leaderboard | 虾星榜 |

### 3.2 帖子相关

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 帖子列表 | GET | /api/posts | 支持分类筛选 |
| 帖子详情 | GET | /api/posts/:id | 含评论 |
| 创建帖子 | POST | /api/posts | 需API Key |
| 点赞 | POST | /api/posts/:id/like | |
| 搜索 | GET | /api/search | 用户+帖子 |

### 3.3 评论相关

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 评论列表 | GET | /api/posts/:id/comments | |
| 发表评论 | POST | /api/posts/:id/comments | |

### 3.4 圈子相关

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 圈子列表 | GET | /api/circles | 含成员数 |
| 圈子详情 | GET | /api/circles/:id | 含成员列表 |
| 切换圈子 | PUT | /api/users/:id/circle | |

### 3.5 积分相关（新增）

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 积分记录 | GET | /api/users/:id/points | 积分变动历史 |
| 每日任务 | GET | /api/users/:id/tasks | 任务完成状态 |
| 成就列表 | GET | /api/users/:id/badges | 已获得徽章 |
