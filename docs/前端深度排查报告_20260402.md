## ⚠️ 可能的问题

### 问题 1：浏览器缓存

**可能原因**：
- 浏览器缓存了旧版本 HTML/JS
- CSS 文件缓存导致样式问题

**解决方案**：
- 强制刷新（Ctrl+Shift+R）
- 清除浏览器缓存
- 添加版本号参数

---

### 问题 2：JavaScript 执行错误

**可能原因**：
- 浏览器控制台有 JS 错误
- 某个函数未定义
- API 响应格式变化

**排查方法**：
- 打开浏览器控制台（F12）
- 查看 Console 错误
- 查看 Network 请求

---

### 问题 3：CSRF Token 问题

**可能原因**：
- CSRF Token 未正确获取
- Token 过期
- Token 验证失败

**检查点**：
- `/api/csrf-token` 接口
- app.js 中 Token 获取逻辑
- 请求头是否正确包含 Token

---

## 📋 建议排查步骤

### Step 1：浏览器控制台检查

1. 打开帖子详情页
2. 按 F12 打开开发者工具
3. 查看 Console 标签页的错误
4. 查看 Network 标签页的请求

### Step 2：清除缓存

1. Ctrl+Shift+Delete 打开清除缓存
2. 选择"缓存的图片和文件"
3. 清除数据
4. 强制刷新页面（Ctrl+Shift+R）

### Step 3：检查头像路径

1. 右键头像图片
2. 选择"检查"
3. 查看图片 URL
4. 直接在浏览器打开图片 URL

### Step 4：检查 API 请求

1. Network 标签页
2. 筛选 XHR 请求
3. 找到 `/api/posts/:id` 请求
4. 查看 Response 数据
