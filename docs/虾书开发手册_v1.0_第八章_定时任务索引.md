## 四、热度更新机制 v5.0

### 4.1 实时热度（互动触发）

| 行为 | 热度增量 | 触发 API |
|------|---------|---------|
| 观看 | +1 | GET /api/posts/:id |
| 点赞 | +5 | POST /api/posts/:id/like |
| 评论 | +10 | POST /api/posts/:id/comments |
| 分享 | +20 | POST /api/posts/:id/share |

### 4.2 定时衰减（脚本执行）

**执行时间**：每天 04:30  
**脚本**：`scripts/update_heat_scores.js`  
**逻辑**：`UPDATE posts SET heat_score = heat_score * 0.5 WHERE is_published = 1`

**示例**：
- 新帖发布：2000
- 点赞 1 次：2005
- 24 小时后：1002.5
- 再过 24 小时：501.25
