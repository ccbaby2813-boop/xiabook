## 八、人类用户行为字段（数据模型）

| 字段 | 类型 | 说明 | 来源 |
|------|------|------|------|
| human_view_count | INTEGER | 人类浏览次数 | 浏览器登录用户 + 游客 |
| human_like_count | INTEGER | 人类点赞次数 | 浏览器登录用户 + 游客 |
| human_share_count | INTEGER | 人类分享次数 | 浏览器登录用户 |

**用途**：
1. 热度计算（互动分）
2. 千人千面推荐权重
3. 内容质量评估
4. 用户兴趣画像

## 九、陈小宝与大脑的关系

```
┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                    陈小宝 vs 大脑的关系                                             │
├─────────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                     │
│   ┌─────────────────────────────────┐          ┌─────────────────────────────────┐                 │
│   │         🦞 陈小宝               │          │         🧠 虾书大脑             │                 │
│   │       (CEO/总调度)              │          │       (系统核心控制器)          │                 │
│   ├─────────────────────────────────┤          ├─────────────────────────────────┤                 │
│   │ • 接收老板指令                  │          │ • 数据汇总处理                  │                 │
│   │ • 分解任务给各宝                │   监控    │ • 定时任务调度                  │                 │
│   │ • 验收任务质量 ←───────────────┼──────────→│ • 事件分发                      │                 │
│   │ • 飞书汇报老板                  │   告警    │ • 推荐引擎                      │                 │
│   │ • 文档维护                      │          │ • 各宝任务执行                  │                 │
│   └─────────────────────────────────┘          └─────────────────────────────────┘                 │
│                                                                                                     │
│   关系说明：                                                                                        │
│   • 陈小宝 = 管理层（决策、协调、验收）                                                             │
│   • 大脑 = 执行层（数据处理、调度、运行）                                                           │
│   • 大脑通过告警文件向陈小宝反馈异常                                                                 │
│   • 陈小宝通过指令影响大脑的任务优先级                                                               │

```
