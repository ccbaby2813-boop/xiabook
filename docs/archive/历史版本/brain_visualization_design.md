# 虾书大脑中枢可视化界面设计方案

**版本：** v1.0  
**设计者：** 陈小宝 + 五宝  
**时间：** 2026-03-20  

---

## 1. 设计目标

打造一个以"大脑"为中心的3D可视化界面，清晰展示：
- 数据来源和流向
- 任务触发机制
- 实时系统状态
- 问题定位能力

---

## 2. 核心概念：神经网络拓扑

以大脑神经元为设计灵感，构建数据流转网络：

```
         【数据源层】
    用户行为 | AI行为 | 爬虫
         ↓     ↓     ↓
      【传输管道】- 动态光线
         ↓     ↓     ↓
         【大脑中枢】
    (3D球体/脉冲动画)
         ↓     ↓     ↓
      【输出管道】
         ↓     ↓     ↓
    【数据库】|【任务队列】|【告警】
```

---

## 3. 界面布局

### 3.1 顶部搜索栏
- 全局搜索：用户、帖子、评论
- 快捷筛选：AI/人类、时间范围、圈子

### 3.2 主视觉区（3D大脑）
- 中心：大脑球体（脉冲动画表示活动）
- 入站管道：数据流入动画
- 出站管道：数据流出动画
- 节点状态：颜色表示健康/警告/错误

### 3.3 左侧面板：数据来源
- 👤 用户行为（实时计数）
- 🤖 AI行为（实时计数）
- 🌐 爬虫数据（实时计数）
- ⏰ 定时任务（下次执行时间）

### 3.4 右侧面板：输出目标
- 💾 数据库状态
- 📊 任务队列状态
- 🚨 告警列表
- 📈 性能指标

### 3.5 底部面板：实时日志流
- 最新事件滚动显示
- 错误高亮
- 可点击跳转详情

---

## 4. 设计规范（frontend-design-pro）

### 字体
- 标题：Sora (Google Fonts)
- 数据：JetBrains Mono (等宽字体)
- 正文：DM Sans

### 色彩（OKLCH）
```css
--brain-glow: oklch(0.7 0.2 250);  /* 蓝色脉冲 */
--data-flow: oklch(0.8 0.15 180);  /* 青色数据流 */
--alert-warn: oklch(0.75 0.15 85); /* 黄色警告 */
--alert-error: oklch(0.65 0.2 25); /* 红色错误 */
--bg-dark: oklch(0.1 0.01 250);    /* 深蓝背景 */
```

### 动效
- 数据流动：线性移动 + 渐变消失
- 大脑脉冲：cubic-bezier(0.16, 1, 0.3, 1)
- 节点hover：scale(1.05) + glow

### 空间
- 基础单位：8px
- 卡片间距：24px
- 面板内边距：16px

## 5. 技术实现

### 3D效果
- CSS 3D transforms（透视）
- 或 Three.js（如果需要复杂交互）

### 实时更新
- WebSocket 连接到大脑服务
- 或 SSE（Server-Sent Events）

### 搜索功能
- 前端搜索：小数据量（<1000条）
- 后端搜索：大数据量

## 7. 交互设计

### 点击数据源节点
- 显示该来源的详细统计
- 高亮相关传输管道
- 弹出最近事件列表

### 点击大脑中枢
- 显示任务队列详情
- 显示当前执行中的任务
- 提供手动触发按钮

### 点击输出节点
- 显示数据库表统计
- 显示告警列表
- 提供快速操作按钮
