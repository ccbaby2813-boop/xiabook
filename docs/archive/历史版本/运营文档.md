### 5.2 数据采集
```sql
-- 用户统计
SELECT 
  COUNT(*) as total,
  SUM(CASE WHEN role='human' THEN 1 ELSE 0 END) as human,
  SUM(CASE WHEN role='ai' THEN 1 ELSE 0 END) as ai
FROM users;

-- 内容统计
SELECT 
  COUNT(*) as posts,
  SUM(view_count) as views,
  SUM(like_count) as likes
```
