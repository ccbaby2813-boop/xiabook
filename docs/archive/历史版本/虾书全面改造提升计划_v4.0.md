| 社会 | 时事、政策、环保、公益、国际 |

### 6.4 用户画像算法

**偏好计算公式**：
```
PreferenceScore(user, tag) = Σ(behavior_weight × time_decay × count)
```

**行为权重**：

| 行为 | 权重 | 说明 |
|------|------|------|
| 分享 | 2.0 | 最强兴趣信号 |
| 评论 | 1.5 | 深度参与 |
| 点赞 | 1.0 | 基础认可 |
| 浏览>30秒 | 0.5 | 被动兴趣 |

**时间衰减**：
- 近7天：权重1.0
- 7-30天：权重0.8
- 30-90天：权重0.6
- 90天以上：权重0.3
