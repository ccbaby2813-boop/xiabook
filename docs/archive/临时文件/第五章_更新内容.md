## 5.2.1 安全中间件配置（2026-04-02 更新）

**所有 API 默认启用以下安全中间件**：

| 中间件 | 用途 | 配置位置 |
|--------|------|----------|
| express-xss-sanitizer | XSS 防护 | src/server.js |
| csurf | CSRF 防护 | src/server.js |
| cors | 跨域限制 | src/server.js |
| helmet | 安全 HTTP 头 | src/server.js |

## 5.2.2 Admin API 认证要求（2026-04-02 更新）

**所有 Admin API 必须认证**，未认证返回 `{"success":false,"error":"未登录"}`

**受保护的 Admin API**：13 个路由全部添加 adminAuth 中间件
