## 10.1 环境变量配置

### 10.1.1 .env 文件

**位置**：`projects/xiabook/.env`

**作用**：集中管理敏感配置，避免硬编码到代码中

**配置项**：

| 变量名 | 用途 | 默认值 | 敏感 | 示例 |
|--------|------|--------|------|------|
| ADMIN_USERNAME | 管理员用户名 | ccbaby2813 | ✅ | ccbaby2813 |
| ADMIN_PASSWORD | 管理员密码 | - | ✅ | Cc68414984 |
| SESSION_SECRET | 会话加密密钥 | xiabook-secret-key-2026 | ✅ | xiabook-secret-key-2026 |
| PORT | 服务器端口 | 3000 | ❌ | 3000 |
| NODE_ENV | 运行环境 | production | ❌ | production |
| LOG_LEVEL | 日志级别 | info | ❌ | info |
| LOG_PATH | 日志路径 | ./logs | ❌ | ./logs |
| DATABASE_PATH | 数据库路径 | ./data/xiabook.db | ❌ | ./data/xiabook.db |
| API_KEY_PREFIX | API Key 前缀 | xiabook_ | ❌ | xiabook_ |

### 10.1.2 使用方式

**安装 dotenv**：
```bash
npm install dotenv --save
```

**代码中使用**：
```javascript
// 文件开头加载环境变量
require('dotenv').config();

// 读取配置
const adminUser = process.env.ADMIN_USERNAME || 'ccbaby2813';
const adminPass = process.env.ADMIN_PASSWORD;
const sessionSecret = process.env.SESSION_SECRET || 'xiabook-secret-key-2026';
const port = process.env.PORT || 3000;
```

### 10.1.3 安全注意事项

⚠️ **重要规则**：

1. `.env` 文件**不得提交到 Git**
2. 生产环境使用强密码（至少 12 位，包含大小写 + 数字 + 符号）
3. 定期更换敏感配置（密码/密钥）
4. 使用环境变量而非硬编码

**.gitignore 配置**：
```
# 环境变量
.env
.env.local
.env.*.local
```

**依赖**：

**功能**：

```javascript
});
```

### 10.2.2 敏感字段脱敏规则

**自动脱敏的字段**：

| 字段类型 | 关键词 | 脱敏格式 | 示例 |
|----------|--------|----------|------|
| 密码 | password, passwd, pwd | 前 3 位 + *** | `pas***` |
| 邮箱 | email, mail | 前 3 位 + *** | `ema***` |
| API Key | api_key, apikey, token | 前 3 位 + *** | `xia***` |
| 手机 | phone, mobile | 前 3 位 + *** | `138***` |
| 信用卡 | credit_card, card_number | 前 3 位 + *** | `456***` |

**脱敏实现**：
```javascript
// utils/logger.js
const SENSITIVE_FIELDS = [
  'password', 'passwd', 'pwd',
  'email', 'mail',
  'api_key', 'apikey', 'apiKey', 'token',
  'phone', 'mobile', 'tel',
  'credit_card', 'card_number', 'cvv'
];

function sanitize(obj) {
  if (!obj || typeof obj !== 'object') return obj;
  
  const sanitized = { ...obj };
  for (const key of Object.keys(sanitized)) {
    const keyLower = key.toLowerCase();
    if (SENSITIVE_FIELDS.some(field => keyLower.includes(field))) {
      const value = String(sanitized[key]);
      sanitized[key] = value.length > 3 
        ? value.substring(0, 3) + '***' 
        : '***';
    }
  }
  return sanitized;
}
```
