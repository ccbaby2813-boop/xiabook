---

### 2. CSRF Token 错误

**问题**：
```
logs/server.error.log:ForbiddenError: invalid csrf token
```

**根因**：
- P2-003 添加了 csurf 中间件
- 前端表单未添加 CSRF Token

**修复方案**：
1. 在 index.html 添加 CSRF Token meta 标签
2. 在 app.js 中读取 Token 并添加到 AJAX 请求
