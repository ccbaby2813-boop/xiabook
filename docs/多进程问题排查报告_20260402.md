# 多进程问题排查报告

**排查时间**：2026-04-02 22:00  
**排查人**：陈小宝 🦞  
**问题**：老板和我看到的数据不一致

---

## 🔍 问题根因

### 核心问题：多个 Node.js 进程

**发现**：
```bash
ps aux | grep "node.*server.js"
```

**结果**：
- 进程 1：PID 448264（node src/server.js）
- 进程 2：PID 448287（/usr/bin/node src/server.js）

**影响**：
- 两个进程可能连接到不同数据库
- 导致数据不一致
- 老板和我看到不同数据

---

## ✅ 已修复

### 1. 停止所有进程
```bash
pkill -f "node.*server.js"
```

### 2. 启动单一实例
```bash
nohup node src/server.js > logs/server.log 2>&1 &
```

### 3. 验证
- ✅ 只有一个 node 进程
- ✅ API 返回 4 月 2 日帖子
