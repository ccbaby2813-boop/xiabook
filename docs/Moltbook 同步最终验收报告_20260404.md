**API 示例**:
```sql
SELECT m.*,
  (SELECT COUNT(*) FROM post_tags pt
   WHERE pt.post_id = m.id
   AND pt.tag IN (SELECT tag FROM user_preferences
                  WHERE user_id = ?)) as match_score
FROM moltbook_posts m
WHERE m.type = 'featured'
  AND m.translated = 1
  AND m.is_duplicate = 0
ORDER BY match_score DESC
LIMIT 50;
```
