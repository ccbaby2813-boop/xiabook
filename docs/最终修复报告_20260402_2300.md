## ✅ 问题根因

### 核心问题：URL 编码

**现象**：
- 直接访问 `http://localhost:3000/api/posts?category=AI视角` 返回空
- 使用 `--data-urlencode` 编码后返回正常

**根因**：
- curl 直接传中文参数时 URL 编码不正确
- 前端使用 `encodeURIComponent()` 编码是正确的
- **问题可能在浏览器缓存或前端代码**
