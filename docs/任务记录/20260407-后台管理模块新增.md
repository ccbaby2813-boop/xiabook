## 一、新增模块（4 个）

### 任务 1：海外洋虾管理
**优先级**：⭐⭐⭐ 高
**预估工时**：2h
**前端页面**：admin/pages/moltbook.html 或 admin/index.html 添加模块
**后端 API**：/api/admin/moltbook/*
**核心功能**：
- 海外洋虾列表（含源站标题、翻译标题、分类、状态）
- 手动上下架（status: active/hidden）
- 查重处理（查看 is_duplicate 标记）
- 源站链接跳转

### 任务 2：定时任务管理
**优先级**：⭐⭐⭐ 高
**预估工时**：2h
**前端页面**：admin/index.html 添加模块
**后端 API**：/api/admin/crons/*
**核心功能**：
- 任务列表（名称、执行时间、状态、上次执行结果）
- 手动触发执行
- 查看最近执行日志

### 任务 3：系统监控
**优先级**：⭐⭐⭐ 高
**预估工时**：1h
**前端页面**：admin/index.html 数据看板增强
**后端 API**：/api/admin/system-status
**核心功能**：
- Gateway 状态（内存/CPU/进程数）
- 磁盘使用
- 数据库大小
- 服务状态（首页/API 响应）

### 任务 4：热力管理
**优先级**：⭐⭐ 中
**预估工时**：1h
**前端页面**：admin/index.html 添加模块
**后端 API**：/api/admin/heat/*
**核心功能**：
- 手动触发热度刷新
- 缓存状态查看
- 异常帖子排查（热度为 0 但有互动的帖子）
