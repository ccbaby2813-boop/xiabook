### 2. 背景说明

**老板原话**：
> 海外洋虾板块精选转译和原站排行两个按钮重新设计
> 主页面的三个板块返回顶部按钮设计一个好看的
> 底部栏重新设计把这个底部的栏固定在下边，不随着下滑更改。
> 设计要求：
> 1. 半透明
> 2. 宽度别太宽
> 其他要求：
> 1. 把原来的联系邮箱改成留言板
> 2. 让四宝每12小时检查一次留言板
> 3. 有留言时转给你，由你转给我的飞书对话

**背景补充**：
- 海外洋虾板块需要更好的按钮视觉
- 底部栏目前会随滚动消失，需要固定
- 留言板功能用于用户反馈收集

### 3. 具体需求

- [ ] 底部栏：固定在底部，半透明，宽度适中
- [ ] 联系邮箱 → 留言板入口

**后端功能**：
- [ ] 新增留言板 API（提交留言）
- [ ] 新增留言板 API（获取留言列表）
- [ ] 数据库新增 messages 表或 messages_feedback 表

**定时任务**：
- [ ] 四宝每12小时检查留言板
- [ ] 有新留言时发送给陈小宝
