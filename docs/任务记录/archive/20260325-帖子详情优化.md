### 2. 老板原话

> 你能不能把那个帖子详情那个弹出页取消掉？点击卡片弹出的就是直接跳转的它的那个独立页面，然后点击返回按钮。返回之后还能记忆到他刚才那个下拉，拉到的那个位置。就是你不要直接就返回主页的顶部了，要返回他那个下拉的记忆点。

### 3. 需求分析

| 当前状态 | 问题 |
|---------|------|
| 点击卡片 → 弹出层 | 显示样式与独立页面不一致 |
| 分享链接 → 独立页面 | 样式不同，体验割裂 |
| 返回主页 → 顶部 | 丢失浏览位置 |

### 4. 解决方案

1. **取消弹出层**：点击卡片直接跳转 `/post/:id`
2. **记住滚动位置**：使用 `sessionStorage` 保存滚动位置
3. **返回时恢复**：页面加载时检查是否有保存的位置

### 5. 技术实现

```javascript
// 点击卡片 → 跳转独立页面
card.onclick = () => {
  sessionStorage.setItem('scrollPos', window.scrollY);
  window.location.href = `/post/${postId}`;
};

// 返回主页时恢复位置
window.onload = () => {
  const pos = sessionStorage.getItem('scrollPos');
  if (pos) {
    window.scrollTo(0, parseInt(pos));
    sessionStorage.removeItem('scrollPos');
  }
};
```
