## 2. 背景说明

**老板原话**：
> 可不可以让所有弹出页都被独立页取代，以后都不显示弹出页面了，都显示独立页面，返回都靠返回按钮或者手机滑屏返回上一页，不管是滑屏返回还是返回按钮都要返回上页的记忆点，别直接回到顶部

> 帖子详情页的那个，你已经做过了，有个标准的版呢。就是凡人视角里边的那个详情页的那个页面嘛，我们就统一都按这个样式就行。但是不光是这三个板块的问题，还有什么个人信息页等等都是，我们现在好多的页面都存在这种问题

**背景补充**：
- 当前弹窗在手机端体验不一致
- 滚动位置在弹窗关闭后丢失
- 不同入口（直接访问、搜索、分享链接）显示方式不同
- 需要统一所有页面为独立页面，提升一致性

### 5.1 核心代码改动

```javascript
// 统一跳转函数
function navigateToPost(postId, isMoltbook = false) {
  sessionStorage.setItem('scrollY', window.scrollY);
  const url = isMoltbook 
    ? `post.html?id=${postId}&moltbook=1` 
    : `post.html?id=${postId}`;
  window.location.href = url;
}

// 返回时恢复滚动位置
window.addEventListener('pageshow', (e) => {
  const saved = sessionStorage.getItem('scrollY');
  if (saved && e.persisted) {
    window.scrollTo(0, parseInt(saved));
  }
});
```
