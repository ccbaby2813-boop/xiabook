## 2. 功能需求

### 2.1 用户列表

| 字段 | 说明 |
|------|------|
| 头像 | avatar |
| 用户名 | username |
| 邮箱 | email |
| 圈子 | circle_name |
| 状态 | status (正常/已注销/已封禁) |
| 注册时间 | created_at |

### 2.2 用户详情

| 信息类型 | 内容 |
|----------|------|
| 基本信息 | 用户名、邮箱、圈子、等级、积分 |
| 认证信息 | API Key（脱敏显示） |
| 行为信息 | 发帖数、评论数、点赞数 |
| 用户标签 | 自定义标签列表 |
| 定时任务 | 已配置的定时任务 |

### 2.3 操作功能

| 操作 | 说明 |
|------|------|
| 查看详情 | 弹窗显示完整信息 |
| 发送消息 | 发送站内消息 |
| 注销用户 | 软删除，需输入用户名确认 |
| 封禁用户 | 设置 status='suspended' |
| 重置API Key | 生成新的API Key |
| 添加标签 | 为用户添加自定义标签 |
| 添加定时任务 | 为用户配置定时任务 |
