# P1-002 登录保护优化方案

**分析时间**：2026-04-02 10:30  
**优先级**：P0（安全升级）

---

## 🔍 当前修复的问题

### 问题 1：内存存储（严重）
```javascript
const loginAttempts = new Map(); // ❌ 重启后丢失
```

**风险**：
- 服务器重启后计数清零
- 攻击者可等待重启后继续攻击
- 集群部署时各节点计数不同步

### 问题 2：无告警
- 被攻击时无人知晓
- 无法及时发现安全事件

### 问题 3：无解锁机制
- 管理员真忘记密码会被锁定
- 需要手动重启服务器解锁

### 问题 4：只保护 Admin
- 用户登录接口无保护
- 普通用户也可能被暴力破解

---

## ✅ 优化方案

### 1. 改用数据库存储（持久化）

```sql
-- 新增登录失败记录表
CREATE TABLE IF NOT EXISTS login_attempts (
  ip TEXT PRIMARY KEY,
  count INTEGER DEFAULT 0,
  last_attempt DATETIME,
  locked_until DATETIME
);
```

```javascript
// 检查登录失败次数
function checkLoginAttempts(ip) {
  return new Promise((resolve) => {
    db.get('SELECT * FROM login_attempts WHERE ip = ?', [ip], (err, row) => {
      if (!row) {
        resolve({ count: 0, locked: false });
        return;
      }
      
      // 检查是否已解锁
      if (row.locked_until && new Date(row.locked_until) < new Date()) {
        db.run('UPDATE login_attempts SET count = 0, locked_until = NULL WHERE ip = ?', [ip]);
        resolve({ count: 0, locked: false });
        return;
      }
      
      resolve({ 
        count: row.count, 
        locked: row.locked_until && new Date(row.locked_until) > new Date() 
      });
    });
  });
}

// 记录失败
function recordLoginAttempt(ip, success) {
  if (success) {
    db.run('DELETE FROM login_attempts WHERE ip = ?', [ip]);
  } else {
    db.run(`
      INSERT INTO login_attempts (ip, count, last_attempt, locked_until)
      VALUES (?, 1, datetime('now'), datetime('now', '+15 minutes'))
      ON CONFLICT(ip) DO UPDATE SET
        count = count + 1,
        last_attempt = datetime('now'),
        locked_until = CASE 
          WHEN count >= 4 THEN datetime('now', '+15 minutes')
          ELSE locked_until 
        END
    `, [ip]);
  }
}
```

### 3. 添加管理员解锁接口

```javascript
// POST /api/admin/unlock-ip
router.post('/admin/unlock-ip', adminAuth, async (req, res) => {
  const { ip } = req.body;
  db.run('DELETE FROM login_attempts WHERE ip = ?', [ip], (err) => {
    res.json({ success: true, message: `IP ${ip} 已解锁` });
  });
});
```
