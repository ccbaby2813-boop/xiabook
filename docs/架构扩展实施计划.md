- [ ] P2-007 Redis Session

**实施步骤**：

#### 1. Redis Session（P2-007）

```bash
# 安装 Redis
npm install connect-redis express-session

# 配置
const RedisStore = require('connect-redis').default;
const redis = require('redis');
const redisClient = redis.createClient({ url: 'redis://localhost:6379' });

app.use(session({
  store: new RedisStore({ client: redisClient }),
  secret: process.env.SESSION_SECRET,
  resave: false,
  saveUninitialized: false,
  cookie: { secure: false, httpOnly: true, maxAge: 24*60*60*1000 }
}));
```

**预计工时**：60 分钟

#### 2. 负载均衡（P2-009）

**方案 A：nginx 负载均衡**

```nginx
upstream xiabook {
  server localhost:3000;
  server localhost:3001;
  server localhost:3002;
}

server {
  listen 80;
  location / {
    proxy_pass http://xiabook;
  }
}
```

**方案 B：Docker Swarm**

```yaml
version: '3.8'
services:
  xiabook:
    build: .
    deploy:
      replicas: 3
      resources:
        limits:
          cpus: '0.5'
          memory: 512M
```

**预计工时**：60 分钟

## P2-033 扩展性限制分析

### 当前限制

| 组件 | 限制 | 瓶颈 |
|------|------|------|
| Node.js | 单线程 | CPU 密集型任务 |
| SQLite | 单文件写入 | 并发写入 |
| 内存 Session | 单节点 | 无法水平扩展 |
| 文件系统 | 单服务器 | 存储空间 |

### 解决方案

| 限制 | 解决方案 | 实施时间 |
|------|----------|----------|
| Node.js 单线程 | Cluster 模式/多进程 | 用户>5 万 |
| SQLite 并发 | PostgreSQL | 用户>10 万 |
| 内存 Session | Redis Session | 用户>5 万 |
| 文件系统 | 对象存储（OSS/S3） | 用户>10 万 |
