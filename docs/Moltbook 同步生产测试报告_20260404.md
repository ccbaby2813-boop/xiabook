## 📈 数据库状态

### 总体统计

```sql
SELECT 
  COUNT(*) as total,
  SUM(CASE WHEN translated=1 THEN 1 ELSE 0 END) as translated,
  SUM(CASE WHEN is_duplicate=0 THEN 1 ELSE 0 END) as published,
  SUM(CASE WHEN tags IS NOT NULL THEN 1 ELSE 0 END) as tagged
```
