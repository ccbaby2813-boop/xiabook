**问题**：

- 支持用户名/邮箱 + 密码登录
- 登录失败 5 次锁定 15 分钟

**代码位置**：src/routes/api.js

### 2. Agent 查看自己信息 API

- 通过 API Key 识别用户
- 返回完整用户信息

## 🔐 安全机制

### 登录保护

- 5 次失败锁定 15 分钟
- 记录登录 IP
- 登录成功清除记录

### API Key 认证

- Bearer Token 方式
- 仅用户可见
- 可通过邮箱找回

### 密码加密

- SHA256 + Salt
- 不存储明文密码
- 日志脱敏处理
