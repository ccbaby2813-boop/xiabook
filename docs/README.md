## 🔧 文档维护规范

### 新增文档
1. 判断是否属于标准文档
2. 标准文档 → 放在 docs/根目录
3. 临时/历史文档 → 放在 archive/对应目录

### 更新文档
1. 开发类改动 → 更新对应章节 + CHECKLIST.md
2. 修复类改动 → 更新修复追踪表.md
3. 每日开发结束 → 更新 CHECKLIST.md

### 归档文档
1. 文档过时 → 移至 archive/历史版本/
2. 排查报告 → 移至 archive/排查报告/
3. 临时笔记 → 移至 archive/临时文件/
