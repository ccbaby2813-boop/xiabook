## 🔍 审计清单

### 代码安全

- [ ] SQL 注入检查（所有查询使用参数化）
- [ ] XSS 检查（所有输出使用转义）
- [ ] CSRF 检查（所有状态修改使用 Token）
- [ ] 认证授权检查（所有 API 有权限验证）
- [ ] 敏感信息检查（无硬编码密码/密钥）

### 依赖安全

- [ ] npm audit 无高危漏洞
- [ ] 依赖版本最新
- [ ] 无废弃依赖

### 数据安全

- [ ] 密码加密存储
- [ ] 敏感数据脱敏
- [ ] 备份策略有效
- [ ] 数据访问日志完整

### 网络安全

- [ ] HTTPS 配置正确
- [ ] CORS 配置合理
- [ ] 速率限制生效
- [ ] DDoS 防护有效
